# IA para el desarrollo de aplicaciones web

Proyecto académico para el curso de **Diseño Web** de USAC - EFPEM.

## Autor

**José Francisco Portales Solval**

## Objetivo

Construir un sitio web profesional usando HTML5, CSS3, JavaScript ES6 y Bootstrap 5, integrando contenido sobre inteligencia artificial aplicada al desarrollo web.

## Stack tecnológico

- HTML5
- CSS3
- JavaScript ES6+
- Bootstrap 5 vía CDN
- Font Awesome vía CDN

## Páginas incluidas

1. **Inicio**
   - Presentación del proyecto
   - Datos del estudiante
   - Acceso a las páginas principales
   - Probador de prompts con JavaScript

2. **IA Generativa y LLMs**
   - Comparativa técnica entre GPT, Claude y Gemini
   - Usos principales de los LLMs
   - Ejemplo de prompt efectivo

3. **Agentes de IA**
   - Agentes de IDE
   - Agentes de terminal
   - Agentes autónomos low-code/no-code
   - Flujo de trabajo recomendado

4. **IA para Pruebas y QA**
   - Self-healing tests
   - Visual testing
   - QA tradicional vs IA-driven QA
   - Generación de datos sintéticos con JavaScript

## Prompts utilizados

### Prompt 1: estructura HTML
> Actúa como desarrollador web profesional. Genera una estructura HTML5 semántica para un sitio académico sobre IA aplicada al desarrollo web. Usa Bootstrap 5 y diseño responsive.

### Prompt 2: diseño CSS
> Crea un archivo CSS moderno con colores institucionales tipo USAC/EFPEM, tarjetas con sombra, botones redondeados, diseño responsive y apariencia profesional.

### Prompt 3: contenido académico
> Redacta contenido claro para estudiantes sobre IA Generativa, LLMs, agentes de IA, herramientas de IDE, agentes de terminal, QA con IA, self-healing tests y visual testing.

### Prompt 4: JavaScript
> Genera funciones JavaScript sencillas para evaluar prompts y crear datos sintéticos de prueba usando arreglos y Math.random().

Presentación del proyecto

Este proyecto muestra el uso de herramientas de IA en el desarrollo web, desde modelos generativos hasta agentes capaces de asistir en programación, terminal, pruebas y automatización.

La propuesta mantiene una estructura sencilla, ordenada y responsive para cumplir con los requisitos del curso de Diseño Web.

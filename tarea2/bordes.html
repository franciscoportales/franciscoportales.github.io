¿Qué es un borde?

Un borde es la línea que rodea un elemento. En CSS puedo controlar: tipo (solid, dashed, double), color y grosor. También puedo hacer bordes redondeados con border-radius.

Ejemplos

Borde solid, 3px, color negro

Borde dashed, 3px, color azul

Borde double, 6px, color rojo

Borde redondeado con border-radius: 30px
.borde1{ border: 3px solid #222; }
.borde2{ border: 3px dashed #1e88e5; }
.borde3{ border: 6px double #e53935; }
.bordeRedondo{ border-radius: 30px; }

Estructura básica de una regla CSS

Selector + Propiedad + Valor

p {
  color: blue;
  font-size: 18px;
}
  • Selector: p
  • Propiedad: color, font-size
  • Valor: blue, 18px

¿Qué es el modelo de cajas (box model) en CSS?

El modelo de cajas explica cómo se calcula el espacio que ocupa un elemento: contenido + padding + border + margin.

Caja ejemplo (tiene margin, padding y border) 
.caja{
  margin: 10px;
  padding: 12px;
  border: 3px solid #1e88e5;
}

Elementos de línea y elementos de bloque

Bloque: ocupan todo el ancho disponible (ej: div, p, section).
Línea: ocupan solo lo que mide su contenido (ej: span, a, strong).

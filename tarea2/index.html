¿Qué es CSS y para qué sirve?

CSS (Cascading Style Sheets) es un lenguaje que sirve para dar estilo a las páginas web. Con CSS puedo cambiar colores, tamaños, tipografías, posiciones, bordes, sombras y hacer que el sitio se vea ordenado y atractivo.

Formas de aplicar CSS (con ejemplo)

a) En línea (inline)

Esto es CSS en línea (dentro del atributo style). 

<p style="color:#1e88e5; font-weight:bold;">Texto</p>

b) Interna (en la misma página)

Ejemplo: se escribe dentro de <style> en el <head>.

<head>
  <style>
    h2 { color: green; }
  </style>
</head>

c) Externa (archivo .css)

Esta página usa CSS externo con css/estilos.css.

<link rel="stylesheet" href="css/estilos.css">

¿Qué es un comentario y cómo se escribe en CSS?

Un comentario es un texto que el navegador ignora; sirve para explicar el código o dejar notas. 

/* Esto es un comentario en CSS */

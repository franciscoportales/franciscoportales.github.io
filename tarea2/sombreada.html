Propiedad text-shadow

Sirve para dar sombra al texto. Se define como: text-shadow: x y blur color;

Este texto tiene sombra (text-shadow)

.textoSombra{
  text-shadow: 2px 2px 4px rgba(0,0,0,0.35);
}

Propiedad box-shadow

Sirve para dar sombra a cajas (div, article, etc.): box-shadow: x y blur color;

Esta caja tiene sombra (box-shadow) 
.cajaSombra{
  box-shadow: 0 8px 18px rgba(0,0,0,0.18);
}
